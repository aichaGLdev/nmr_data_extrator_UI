Services

Not just signal processing

CPMolgen also handles chemical structures. They can either be imported from a MDL Molfile, copy pasted directly in the molecule panel or drawn.

1D spectra

CPMolgen accepts 1D spectras. For 1D spectra it can either be a FID or a fourrier transformed spectrum. Currently, only FT 2D spectra are allowed.

Smart peak picking

CPMolgen includes an advanced peak picking detection for 1D spectras and is able to generate the NMR string required for publication or patent.

Great user experience

To provide an optimal user experience, the spectra processing is efficiently performed within the web browser

Advanced Integration

CPMolgen offers seamless integration with other chemical informatics tools and databases, allowing for enhanced data analysis and management.
